# yalla-farm-front

Отдельный frontend для Yalla Farm на Next.js + TypeScript + Tailwind + Zustand + Redux + SignalR.

## Запуск

1. Скопируйте `.env.example` в `.env.local`.
2. Установите зависимости: `npm install`
3. Запуск dev: `npm run dev`

По умолчанию frontend ожидает API на `http://localhost:5000`.
